E6 : Situations professionnelles

Situation 1 : Sauvegarde avec Iperius

Dans le cadre de notre projet d’infrastructure BELLETABLE j’ai été chargé de mettre en place la sauvegarde pour assurer la pérennité des données.

La solution de sauvegarde choisie est : Iperius Backup. Il permet la sauvegarde de PC, serveurs, machines virtuelles, bases de données, serveurs de messagerie et Microsoft 365 sur n'importe quelle destination.

Iperius est installé sur un serveur dédié. Les systèmes concernés par la sauvegarde sont :

  • Le serveur de fichier
  • Le serveur web

Les données seront stockées sur un NAS Synology.

Situation 2 : Supervision avec Prometheus et Grafana

Dans le cadre de notre projet d’infrastructure BELLETABLE, j’ai été chargé de trouver une solution de supervision pour monitorer des serveurs critiques de notre infrastructure.

La solution choisie est Prometheus pour exporter les métriques du serveur vers Grafana. Grafana permet la collecte, la corrélation et la visualisation des données avec des tableaux de bord. Ce sont toutes deux des solutions OpenSource permettant de limiter les coûts.
